# Что это?
Это симулятор бронепробиваемости для танков Второй мировой войны. 
Программа рассчитывает вероятность пробития брони в зависимости от типа танка и расстояния. Данные о бронировании усреднены, а в значении толщины учтена толщина брони.
## Возможности

- Загрузка данных о танках из CSV файла
- Выбор стреляющего танка и цели из 30 моделей (есть возможность масштабировать)
- Расчёт пробития с учётом:
  - Начальной скорости снаряда
  - Массы снаряда
  - Калибра
  - Расстояния до цели
- Специальные формулы для разных типов танков (функции UnregularRechnung и ChaffeRechnung)
- Сохранение результатов в .txt файл

## Как пользоваться?
#### Файл таблицы должен находиться в одной папке с исполняемым файлом
- Посмотреть в таблицу и выбрать номера танка-стрелка и танка-цели
- Поочерёдно ввести их в консоль
- Ввести расстояние (Чем оно больше, тем больше погрешность)

Далее в консоль будут выведены результаты пробития/непробития в трёх проекциях. Также результат будет записан в .txt файл

#### В формулах рассчёта используются разные коэффициенты:
- 0.65 из RegularRechnung - усреднённый коэффициент падения скорости снаряда на дистанции
- 0.28 из UnregularRechnung - коэффициент для Jagdpanther и Panther, связанный с особенностями снаряда
- 0.71 из ChaffeRechnung - коэффициент для M24 Chaffe
